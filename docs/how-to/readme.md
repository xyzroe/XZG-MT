## ğŸ¤ Contribute

The XZG-MT community thrives on contributions! If you've successfully used XZG-MT for a task not covered here, or if you have expertise in a particular area, please share your knowledge by creating a new guide.

### How to Contribute:

1. Fork the [XZG-MT repository](https://github.com/xyzroe/XZG-MT).
2. Create a new Markdown file in the `docs/how-to/` directory (e.g., `your_guide.md`).
3. Follow the structure of existing guides: include an introduction, required hardware, step-by-step instructions, troubleshooting, and a "If the Problem Persists" section.
4. Add emojis to H2 headings for visual appeal.
5. Update this index.md file to include your new guide in the "Existing Guides" list.
6. Submit a pull request with a clear description of your changes.

We appreciate your help in making XZG-MT more accessible to everyone!
